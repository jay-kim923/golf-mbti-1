# **프로젝트 개요**

이 프로젝트는 골퍼들의 성향을 MBTI 형식으로 분석하여, 개인의 플레이 스타일과 개선 방향을 제시하는 웹 애플리케이션입니다. 20개의 설문 문항을 통해 사용자의 성향을 파악하고, 16가지 골퍼 캐릭터 유형 중 하나를 결과로 보여줍니다. 각 캐릭터 유형별로 장점, 단점, 그리고 추천 연습 방법을 제공하여 사용자가 자신의 골프 실력을 향상시키는 데 도움을 주는 것을 목표로 합니다.

# **구현된 기능 및 디자인**

## **v2.1 (현재 버전)**

*   **단일 파일 구조:** HTML, CSS, JavaScript를 `index.html` 단일 파일에 통합하여 프로젝트 구조를 단순화했습니다.
*   **직관적인 UI:** 간결하고 사용하기 쉬운 인터페이스를 제공하여 사용자가 테스트에만 집중할 수 있도록 했습니다.
*   **20가지 설문 조사:** 사용자의 골프 성향을 5가지 MBTI 척도(E/I, S/N, T/F, J/P)에 따라 파악하기 위한 20개의 질문을 제공합니다.
*   **4가지+ 기본 캐릭터:** 테스트 결과에 따라 '필드의 전략 지휘관'(ESTJ), '감성 몰입형 아티스트'(INFP), '변수 지배자'(ENTP), '데이터 마스터'(ISTJ) 등 4가지 주요 캐릭터와 그 외 결과에 대한 '올라운드 플레이어' 기본 캐릭터 정보를 제공합니다.
*   **결과 분석:** 설문 결과를 바탕으로 MBTI 유형을 결정하고, 해당 유형의 이름, 성향 설명, 장점, 약점, 추천 연습법을 명확하게 보여줍니다.
*   **제휴 문의 폼:** Formspree를 연동하여 사용자가 이메일과 문의 내용을 제출할 수 있는 제휴 문의 폼을 추가했습니다.

# **현재 요청 계획**

사용자의 요청에 따라 `index.html` 하단에 Formspree를 사용한 제휴 문의 폼을 추가했습니다. `blueprint.md` 파일을 생성하여 프로젝트의 전체 내용과 새로운 기능 추가 사항을 기록했습니다.
